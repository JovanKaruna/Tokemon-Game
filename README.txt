#TOKEMON PRO & LOG
Program ini dibuat untuk memenuhi salah satu Tugas Besar IF2121 Logika Komputasional Teknik Informatika ITB.

##Requirements :
Download SWI-Prolog to use this program

##How To Use :
1. Run main.pl
2. Choose either "start." to start the game or "loadFile(Filename)" if you already have a saved file
3. In the game use "w","a","s","d" to move
4. Use "map." to show the map in the game
5. Use "help." for further explanation
